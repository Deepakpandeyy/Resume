Project : Plain Shooter Game

Description

In this project, I have created a simple but engaging 2D shooter game in which the player has to shoot enemy targets. The game has basic shooting mechanics, anime AI, and score tracking features. Using the Unity engine, I have optimized the game and implemented user-friendly controls.

Features:

Basic Shooting Mechanism:

The player has to target and shoot the animes with the mouse and keyboard.

Anime Eye:

Anime makes random movements and challenges the player.

Scoring System:

Every anime kills earns points and the high score tracking system is implemented.

Minimalist Design:

Simple graphics and clear UI design that puts the focus on gameplay.

Technologies Used:

  • Unity 2D
  • C# for gameplay scripting
  • Basic AI techniques for enemy movement
  • Particle effects for shooting and explosions

Challenge Faced & Solutions

Challenge:

Making the movement of anime AI random and engaging.

Solution:

Finally tuned pathfinding algorithm and random movement patterns that make the gameplay unpredictable and interesting.

Learning & Outcome:

From this project, I understood Unity 2D development and game mechanics deeply. This game is a strong addition to my portfolio that showcases my problem-solving skills, creativity, and coding capabilities.

Game Code Snippet:


// Unity C# script for player movement
using UnityEngine;

public class PlayerMovement : MonoBehaviour
{
public float moveSpeed = 5f;

void Update()
{
float horizontal = Input.GetAxis("Horizontal");
float vertical = Input.GetAxis("Vertical");
Vector3 movement = new Vector3(horizontal, 0, vertical) * moveSpeed * Time.deltaTime;
transform.Translate(movement);
}
}
