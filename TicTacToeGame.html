Projects

Project : Tic Tac Toe Game

Description

In this project, I developed a classic 2D Tic Tac Toe game offering a simple and intuitive gameplay experience. The game is built using Unity, where players can select X or O and make their moves. The game also includes an AI opponent for single-player mode, allowing players to test their skills.

Features:

Multiplayer Mode

Two players can play the game on a single device.

Grid System:

A 3x3 grid where players place their moves, providing an intuitive and clear interface.

Victory Detection:

The game automatically detects if a player has aligned three X's or O's horizontally, vertically, or diagonally.

Reset Game Option:

After a game, players can reset and start a new round.

Simple UI:

Clean and minimalistic interface that makes it easy to understand the game rules.

Technologies Used:

  • Unity (for game engine)
  • C# for gameplay scripting
  • 2D graphics for game elements (Grid, X and O icons)

Challenge Faced & Solutions

Challenge:

Implementing victory conditions and making the AI opponent challenging.

Solution:

Used efficient algorithms to check all rows, columns, and diagonals for a win, and implemented simple logic and randomization to make the AI opponent challenging.

Learning & Outcome:

This project allowed me to gain a solid understanding of game logic, and 2D gameplay development in Unity. It is a great addition to my portfolio, showcasing my logical thinking, problem-solving skills, and ability to design and develop engaging games.

Game Code Snippet:


// Unity C# script for player movement
using UnityEngine;

public class PlayerMovement : MonoBehaviour
{
public float moveSpeed = 5f;

void Update()
{
float horizontal = Input.GetAxis("Horizontal");
float vertical = Input.GetAxis("Vertical");
Vector3 movement = new Vector3(horizontal, 0, vertical) * moveSpeed * Time.deltaTime;
transform.Translate(movement);
}
}
